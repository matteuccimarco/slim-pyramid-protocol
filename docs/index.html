The 10 Levels

Level Name Tokens Purpose
L0 CAPS ~20 Capabilities only - what can you do with this?
L1 META ~50 Identity + metadata - what is this?
L2 NAV ~100 Navigation outline - how is it organized?
L3 INDEX ~200 Structure + entities - what's in it?
L4 SUMMARY ~400 Summaries + key facts - actionable info
L5 CONTENT ~800 Full text by unit - detailed content
L6 STRUCT ~1500 Data schemas - table/list definitions
L7 FULL Variable Complete content - everything
L8 MEDIA_META ~200 Media inventory - what media exists
L9 MEDIA_DESC ~1000 AI descriptions - media understanding

Quick Start

TypeScript 
import { SLIML4, isSLIML4 } from '@slim-protocol/types';

// Fetch content at level 4
const response = await fetch('/api/content?format=slim&level=4');
const payload = await response.json();

// Type-safe consumption
if (isSLIML4(payload)) {
  console.log(payload.keyFacts);     // string[]
  console.log(payload.summaries);    // Record<string, string>
  console.log(payload.actionItems);  // string[] | undefined
}

GET /api/content HTTP/1.1
